Components Context Checklist

Core Controller
- Raspberry Pi 5

Vending Motors
- 28BYJ-48 stepper motor (12V variant for product dispensing)
- ULN2003AN stepper motor driver board (one per tray/lane)
  - ULN2003 IN1..IN4 are driven by MCP23017 GPIO expander outputs
  - ULN2003 board power input is supplied by external 12V PSU for the 12V motor
  - Raspberry Pi and 12V PSU grounds are tied together (common ground)
- 3x MCP23017-E/SP I2C GPIO expander
  - Addresses used: 0x20, 0x21, 0x22
  - Provides independent IN1..IN4 control lines for slots 1..10

RFID Modules
- 1x MFRC522 13.56 MHz RFID reader (shared)
  - Shared for card payment/reload and both door authentication/unlock flows:
    - Restock door
    - Troubleshooting/wiring door

Door Access Hardware
- 2x 12V solenoid locks (Model 102)
  - Restock door
  - Troubleshooting/wiring door

Cash Inputs
- WEIYU Universal Coin Slot (1239/1239A)

Removed from current build
- TB74 bill acceptor
- Allan coin hopper modules and hopper pulse feedback wiring

Vend Confirmation
- IR Break-Beam sensor at product drop path

Software/Database Context Added
- RFID users now support role-based door access via rfid_users.role:
  - customer
  - restocker/staff
  - researcher/troubleshooter
  - admin
- Staff screen now lets operator choose door (restock or troubleshoot) and unlock via RFID role validation.
