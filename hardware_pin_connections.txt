Raspberry Pi 5 GPIO and MCP23017 Connections

IMPORTANT:
- Use a common GND between Raspberry Pi and all external 12V driver supplies.
- Do NOT drive motors/solenoids directly from Pi GPIO.
- ULN2003 controls the motor coils; MCP23017 provides extra control pins for ULN2003 IN1..IN4.

========================================
1) MFRC522 RFID Reader (single shared module)
========================================
Shared SPI lines:
- SCK  -> GPIO11 (Physical pin 23)
- MOSI -> GPIO10 (Physical pin 19)
- MISO -> GPIO9  (Physical pin 21)
- VCC  -> 3.3V
- GND  -> GND

Shared reader control:
- SDA/SS -> GPIO8  (Physical pin 24, CE0)
- RST    -> GPIO5  (Physical pin 29)

RFID flow usage:
- Payment tap
- Card reload tap
- Restock door unlock authorization tap
- Troubleshoot door unlock authorization tap (admin role only)

========================================
2) Stepper Expansion (28BYJ-48 + ULN2003 + MCP23017-E/SP)
========================================
Per ULN2003 board (12V motor variant):
- Motor connector -> ULN2003 motor socket
- ULN2003 V+ / DC+ -> +12V external PSU
- ULN2003 GND / DC- -> PSU GND
- Pi GND -> same PSU GND (common ground)
- MCP23017 GPIO -> ULN2003 IN1, IN2, IN3, IN4

MCP23017 bus wiring:
- Pi GPIO2  (Physical pin 3)  -> SDA
- Pi GPIO3  (Physical pin 5)  -> SCL
- 3.3V -> VDD
- GND  -> VSS
- RESET -> 3.3V through a pull-up resistor (recommended: 10k ohm; acceptable range: 4.7k to 10k)
- A0/A1/A2 set per chip address

Configured expander addresses in software:
- MCP #1: 0x20
- MCP #2: 0x21
- MCP #3: 0x22

Slot to expander pin mapping (main.py default):
- Slot 1:  0x20 GP0..GP3
- Slot 2:  0x20 GP4..GP7
- Slot 3:  0x20 GP8..GP11
- Slot 4:  0x20 GP12..GP15
- Slot 5:  0x21 GP0..GP3
- Slot 6:  0x21 GP4..GP7
- Slot 7:  0x21 GP8..GP11
- Slot 8:  0x21 GP12..GP15
- Slot 9:  0x22 GP0..GP3
- Slot 10: 0x22 GP4..GP7

Notes:
- Runtime selects MCP mode by default on Raspberry Pi.
- To force old direct GPIO stepper mode, set STEPPER_BACKEND=native_gpio before launching.
- MCP logical pin map in tools/tests:
	- GPA0..GPA7 = pins 0..7
	- GPB0..GPB7 = pins 8..15
	- GPB0 specifically equals logical pin 8

========================================
3) Solenoid Locks (12V Solenoid Model 102)
========================================
Songle dual-channel 5V relay module (JD-VCC + VCC type):
- Keep JD-VCC <-> VCC jumper installed for simple single-supply mode.
- Relay VCC/JD-VCC -> Pi 5V (Physical pin 2 or 4)
- Relay GND -> Pi GND (Physical pin 6)
- IN1 -> GPIO16 (Physical pin 36)  [restock lock]
- IN2 -> GPIO20 (Physical pin 38)  [troubleshoot lock]

About the two GND pins and two VCC pins:
- With jumper installed, the two power domains are tied; one GND connection is enough.
- If convenient, both GND pins can be tied to the same ground rail.
- Remove jumper only for split/isolated supply experiments.

Relay contact wiring for each 12V solenoid lock:
- External PSU +12V -> COM (relay channel)
- NO (relay channel) -> Solenoid +
- Solenoid - -> External PSU GND
- Tie external PSU GND to Pi GND (common reference)

Protection and behavior notes:
- Add flyback diode across each solenoid coil:
	stripe/cathode -> solenoid +, anode -> solenoid -
- Most Songle relay modules are active-low:
	IN=LOW energizes relay, IN=HIGH releases relay.
- Keep unlock pulse short (for example 1-3 seconds); many lock solenoids are not meant for long continuous hold.

========================================
4) Cash Input (coin-only)
========================================
Coin acceptor pulse output:
- Pulse OUT -> GPIO19 (Physical pin 35)

Removed hardware in current build:
- Bill acceptor (not used)
- Coin change hopper(s) and hopper pulse feedback lines (not used)

========================================
5) IR Break-Beam Sensor (vend confirmation)
========================================
- Sensor digital OUT -> GPIO26 (Physical pin 37)
- VCC based on module spec (3.3V/5V compatible module recommended)
- GND -> GND
